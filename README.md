# print-pdf
Simple HTMLElement to PDF

## Installation
    npm i print-pdf

## Usage
```typescript
    import { PrintPDF } from 'print-pdf';
    
    new PrintPDF(document.body).printPDF();
```

This library is relatively new so please feel free to leave any suggestions.
